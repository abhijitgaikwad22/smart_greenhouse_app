🚀 ABOUT THE PROJECT 🌱

Smart Greenhouse AI

Intelligent Environment Monitoring & Advisory System for Modern Agriculture

🌿 Project Overview

Smart Greenhouse AI is a cutting-edge web application designed to revolutionize greenhouse farming in Maharashtra. Our platform uses intelligent rule-based algorithms to monitor environmental parameters and provide real-time advisory to farmers.

With support for 20+ major crops of Maharashtra including Tomato, Grapes, Pomegranate, Cotton, Sugarcane, and more, the system analyzes temperature, humidity, soil moisture, and light intensity to generate crop-specific recommendations.

The application features AI-powered crop recommendation, fertilizer guide, weather advisory, and detailed analytics dashboard - all designed with a modern glassmorphism UI and bilingual support (English & Marathi) for Maharashtra farmers.
